## guide of SPA
Input the next command:
        
        npm init -y
1. Install npm and configure everything like as a Json-server, for that:
    
input the command: 

    npm install -D vite


Next, we need to enter in "package.json" and input the next line in scripts:
    
    "scripts": {
        "dev":"vite"
    }

After, we will write this:

    npm run dev
